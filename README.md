## Usage :
```bash
git clone https://github.com/marbenMB/Cub3d.git
cd Cub3d
make bonus
./cub3d_bonus map/path
# example
./cub3d_bonus map/bonus/tech_S.cub
``````
### Keys :
````bash
W : go forward
s : go backward
a : go left
d : go right
<- -> and mouse : camera mouvements
1 : display and undisplay minimap
ESC : close
`````
